Docker——在docker上安装NESSUS扫描器

2016-12-19

搭建自己的武器库
咱们先讲讲如何搭建自己的武器库，不管你是做开发，渗透，还是其他工作，都需要有自己常用和顺手的工具，但是很多时候，鱼和熊掌不可兼得，比如在windows下不能同时安装天境和NESSUS，很多工具我们需要借助虚拟机来完成。不过在你学过docker以后，你会发现，其实这都不是事。本文是讲述一下如何在Ubuntu/kali下安装DOCKER版nessus扫描器。

Docker的安装

这里我们用的是ubuntu系统。所以docker的安装比较简单

sudo apt-get install docker.io

如果你的源没有docker的安装包，请自己添加163源。

然后去下载docker的nessus镜像

sudo docker search nessus

将镜像pull下来：

sudo docker pull 镜像名称

查看一下镜像信息：

启动：

sudo docker run -p 8834:8834 ca0d592f478c (这里的镜像ID换成你自己的)

直接访问https://localhost:8834
